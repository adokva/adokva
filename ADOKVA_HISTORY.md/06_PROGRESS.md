# PROGRESS

Версия документа: 1.0

---

# Назначение

Этот документ фиксирует историю развития проекта ADOKVA.

В отличие от CHANGELOG, который хранит список изменений по версиям, PROGRESS показывает общий путь развития проекта от идеи до текущего состояния.

После завершения каждого крупного этапа данный документ обновляется.

---

# ЭТАП 0 — Идея проекта

Статус: ✅ Завершён

Описание:

Сформирована концепция интерактивной 3D-платформы нового поколения.

Основные цели:

- объединить Землю и Солнечную систему;
- создать единое пространство без переходов между страницами;
- построить масштабируемую архитектуру;
- использовать современные web-технологии.

---

# ЭТАП 1 — Создание проекта

Статус: ✅ Завершён

Выполнено:

- создан проект на Next.js;
- подключён TypeScript;
- подключён React Three Fiber;
- подключён Drei;
- создана базовая структура каталогов;
- настроена среда разработки.

Результат:

Получен рабочий фундамент проекта.

# ТЕКУЩИЙ ЭТАП

- движение планет;
- визуальные эффекты;
