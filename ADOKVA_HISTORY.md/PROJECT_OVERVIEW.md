# Что такое ADOKVA

ADOKVA — это долгосрочный проект интерактивной трёхмерной платформы нового поколения.

Главная идея проекта заключается не в создании очередного 3D-сайта, а в построении единой цифровой среды, объединяющей Землю, космос, Солнечную систему и глобальную навигацию.

Проект разрабатывается постепенно, этап за этапом, с профессиональной архитектурой и полной документацией.

---

# Философия проекта

ADOKVA строится по принципу:

"Каждый новый элемент должен усиливать уже существующую систему, а не усложнять её."

Мы не создаём набор отдельных эффектов.

Мы строим единую экосистему.

---

# Основные принципы разработки

Не писать временный код.

Не создавать дублирование.

Не использовать большие компоненты без необходимости.

Каждая новая возможность должна вписываться в существующую архитектуру.

---

# Документация

Документация считается частью проекта.

Изменение архитектуры сопровождается изменением документации.

Если документация устарела — считается, что проект описан неверно.

---

# Совместная разработка

Проект создаётся совместно человеком и ChatGPT.

Роли разделяются следующим образом.

Разработчик:

• принимает решения;
• тестирует;
• управляет проектом.

ChatGPT:

• проектирует архитектуру;
• пишет код;
• анализирует ошибки;
• помогает сохранять единый стиль проекта;
• следит за целостностью архитектуры.
